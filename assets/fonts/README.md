# Font Download Instructions

## Inter Font Family

Download the Inter font from Google Fonts and extract to `assets/fonts/`:

### Option 1: Manual Download
1. Visit: https://fonts.google.com/specimen/Inter
2. Click "Download family"
3. Extract the `.zip` file
4. Copy these files to `assets/fonts/`:
   - `Inter-Regular.ttf` → Rename to `Inter-Regular.woff2` (or use online converter)
   - `Inter-Medium.ttf` → Rename to `Inter-Medium.woff2`
   - `Inter-SemiBold.ttf` → Rename to `Inter-SemiBold.woff2`

### Option 2: Direct Download (WOFF2)
Use Google Fonts Helper: https://gwfh.mranftl.com/fonts/inter

Select weights: 400, 500, 600
Format: WOFF2
Download and extract to `assets/fonts/`

### Required Files
```
assets/fonts/
├── Inter-Regular.woff2
├── Inter-Medium.woff2
└── Inter-SemiBold.woff2
```

After downloading, the CSS will automatically load these fonts locally.
